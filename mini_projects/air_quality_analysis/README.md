# air_quality_analysis

Análise de qualidade do ar em diferentes cidades e fatores poluentes.

Dataset do Kaggle: [https://www.kaggle.com/sogun3/uspollution](https://www.kaggle.com/sogun3/uspollution)

## Estrutura do projeto
- data/: Dados do projeto (raw, processed, external)
- notebooks/: Notebooks de exploração e análise
- src/: Código modular reutilizável
- app/: Dashboard ou interface interativa
- reports/: Relatórios e visualizações
